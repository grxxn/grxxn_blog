## MDX가 뭐길래?
MDX는 Markdown + JSX의 줄임말입니다.
기존 Markdown처럼 글을 쓰면서 필요한 경우 리액트 컴포넌트를 불러다 쓸 수 있는 기능이 있습니다.
예를 들어 글 중간에 코드블록, 차트, 알림박스 같은 컴포넌트를 바로 삽입할 수 있습니다.
```
# 오늘 배운 것 
React에서 `useState`를 쓸 수 있습니다
<Counter />>
<CustomButton>클릭!</CustomButton>
```
